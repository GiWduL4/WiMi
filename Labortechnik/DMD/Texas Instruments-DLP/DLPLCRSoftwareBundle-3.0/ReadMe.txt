GUI Version 5.0:
---------------------------------------------------------------------------------------------------------------------------
GUI is available in Windows, Linux and MAC versions.

Source code located in "GUI v5.0 - Source" directory

New Features:
* Added mA approximations to LED Current settings
* Added Splash Utility tool to change startup screens on DLP LightCrafter
* Added Additional Information tab 
* Added Multiple Bit and Multiple Color Sequence Settings tab to support automatic import of MBMC sequence, readme,
  and image list files and automatically parse images
* External Streaming PAttern Sequence settings displays the patterns per frame and pattern rate

Bug Fixes:
* GUI on Mac properly resizes
* Removed "Rotate -90 degree" setting from Image Orientation Settings because GUI doesn't allow 
  image data to be sent that would work with the rotated setting
* Moved Output Trigger Settings outside of Pattern Sequence tab to show that the trigger can also 
  be used in the External Streaming Pattern Sequence mode
* Changed HDMI Video tab to HDMI Port tab to better reflect DLPC300 operation in DLP LightCrafter
* Separated Solution and Upgrade tab

---------------------------------------------------------------------------------------------------------------------------
DM365 Software version 5.0
---------------------------------------------------------------------------------------------------------------------------
DM365 DVSDK available from http://www.ti.com/tool/dlplightcrafter

The DM365 build contains two images:
1) lcr_dm365_v5_nand.dat: Boots the device from NAND flash
2) lcr_dm365_v5_sd.dat:   Boots from SD card

## Important Note: The DM365 boot utility(lcr_boot_sd.exe) is not available for MAC or Linux.

New Features:
* Added camera support for Leopard Imaging MT9P031 sensor 

Bug Fixes:
* Fixed startup issue caused by improper NAND initialization
* Fixed output trigger frequency frozen at 30Hz in HDMI mode
