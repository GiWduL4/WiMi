Product name and version number: DLP LightCrafter API Sample Code - Windows v2.0
Company name: Texas Instruments

This file contains important information about LightCrafter sample Code running in Windows. Complete information about DLP® LightCrafter™ is available at http://www.ti.com/tool/dlplightcrafter

Why should I use it?
This VisualStudio C++ project is created using the LightCrafter's sample code. The code shows implementation of  the DM365 command interface to interact with the DLP LightCrafter. This code uses WinSocket2 APIs for TCP communication. The command definitions are described in "DLP® LightCrafter™ DM365 Command Interface Guide" document.

How do I use it?
1. Download and Install the Visual Studio 2008 or later versions.
2. Open the LCR_Cmd_Interface.sln 
3. Build the solution and run the .exe file generated in Release folder.
4. Follow the instructions on the command window

Important Note: The sample source code provided is compatible with DM365 v4.0 or later.
