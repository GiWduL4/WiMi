Compiled with QT5.1.0 and MinGW48_32 using QTCreator 2.8.0

The LightCrafterGUI needs the following dll's in the same directory as the .exe
- Qt5Widgets.dll
- Qt5Network.dll
- Qt5Gui.dll
- Qt5Core.dll
- qwindows.dll (in the same directory as the .exe store this dll within a directory called 'platforms' )
- qminimal.dll (in the same directory as the .exe store this dll within a directory called 'platforms' )
- libwinpthread-1.dll
- libstdc++-6.dll
- libgcc_s_dw2-1.dll
- libEGL.dll
- icuuc51.dll
- icuin51.dll
- icudt51.dll
- mingwm10.dll
